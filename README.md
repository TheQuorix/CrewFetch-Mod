# 🚀 Crew Fetch Mod

**Crew Fetch** is a lightweight utility mod for **Lethal Company** that adds a Linux-style `neofetch` command to your ship's terminal. It provides a quick system overview of your crew, your loot, and your current mission status.

---

### ✨ Features

| Feature | Description |
|---|---|
| **`crewfetch` Command** | Type this in the terminal to see a detailed report of your current session. |
| **Crew Status** | Shows how many employees are still alive and on duty. |
| **Ship Inventory** | Automatically calculates the amount of scrap inside the ship and its total estimated value. |
| **Mission Data** | Displays the current Moon, active Weather, and remaining Credits. |
| **Quota Tracker** | Shows your progress toward the Company Quota and how many days are left. |
| **Help Menu** | Automatically integrates into the `other` command list in the terminal. |

---

### 🖥️ Preview

When you type `crewfetch`, you'll see something like this:

```text
            [ COMPANY QUOTA ]
            Fulfilled: $200 / $1000

```

### 🛠 Installation

1.  Make sure you have [BepInEx](https://thunderstore.io/c/lethal-company/p/BepInEx/BepInExPack/) installed.
2.  Download the `CrewFetch.dll`.
3.  Place the `CrewFetch.dll` into your `Lethal Company/BepInEx/plugins` folder.
4.  Launch the game and enjoy!

---

*   **BepInEx Pack**
*   **Lethal Company** (v45 or newer recommended)
